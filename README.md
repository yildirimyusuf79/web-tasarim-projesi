# Countrypedia

Statik ve çok sayfalı bir ülke tanıtım web projesi.

## Proje Hakkında

Countrypedia, kullanıcının farklı ülkeler hakkında hızlı ve düzenli bilgiye ulaşmasını hedefleyen statik bir web sitesidir.
Proje; ana sayfada dünya haritası üzerinden ülke seçimi, yan menü ile sayfalar arası kolay geçiş ve ülke bazlı detay sayfaları sunar.

## Özellikler

- Çok sayfalı yapı (ana sayfa + konu sayfaları + ülke detay sayfaları)
- Harita üzerinden ülke seçimi
- Sabit üst menü ve açılır yan menü
- Kültür, gelenek, tarih, iklim ve gezi odaklı içerik yapısı
- Mobil uyumlu görünümü destekleyen viewport ayarı
- Zengin görsel içerik klasörü

## Kullanılan Teknolojiler

- HTML5
- CSS3
- Vanilla JavaScript

## Projeyi Çalıştırma

1. Bu depoyu klonlayın:

   ```bash
   git clone https://github.com/yildirimyusuf79/web-tasarim-projesi.git
   ```

2. Proje klasörüne geçin:

   ```bash
   cd web-tasarim-projesi
   ```

3. `index.html` dosyasını tarayıcıda açın.

İsterseniz VS Code `Live Server` eklentisi ile de çalıştırabilirsiniz.
